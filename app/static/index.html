📁 Load Tesla privacy / usage data

Request your data from Tesla (tesla.com → Privacy → Download Your Data). Tesla's ZIP is password-protected, so unzip it first and select the CSVs inside — you can pick several at once (e.g. “Charging Data.csv” + “Vehicle Details.csv”) and they'll be merged. CSV / JSON accepted.

⬆️ Drop a file here or browse
Expected columns

Drives: start_time, end_time, distance (km or miles), duration, start_soc, end_soc, energy_used, avg_speed, outside_temp, start/end_location.

Charges: start_time, end_time, energy_added, charge_type (AC/DC), max_power, start_soc, end_soc, location, cost.

Column names are matched loosely (case / spacing insensitive).

🔗 Link your Tesla account
